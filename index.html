Examples

Required params are only coordinates fields, which can be pre-populated. Hint: drag marker on map.

CSS:

#map { width:320px; height:200px; }

JavaScript:

$("#map").geolocate({
	lat: "#lat",
	lng: "#lng"
});


Fields can be hidden (see page source) and you can pass any additional options for Google Maps API, both for map (defined here) and marker (defined here).

$("#map2").geolocate({
	lat: "#lat2",
	lng: "#lng2",
	mapOptions: {
		mapTypeId: "satellite"
	}
});

Geolocating from address

You can get position from address field (or any number of fields). Location will populate when you change text field and focus out. Hint: try completing address info below.

Note: geocoding is an expensive operation, and there are Google Geocoding API usage limits for non-Premium accounts, so be sure not to do this automatically on every page load.

$("#map3").geolocate({
	lat: "#lat3",
	lng: "#lng3",
	address: ["#address3"]
});
